How It Works

Upload a CSV, run the analysis, and review a structured report. The app profiles columns, validates schema, compares drift against a baseline (optional), and generates actionable recommendations.

  • Inputs: dataset CSV + optional baseline CSV
  • Outputs: KPI summary, charts, agent feedback, AI insights, raw JSON
  • Ideal for: data quality checks, onboarding new datasets, monitoring change

Technical Specs

  • Frontend: static HTML/CSS/JS dashboard
  • Backend: FastAPI + pandas + numpy
  • Workflow: LangGraph pipeline (4 agents)
  • AI: OpenAI Responses API (optional)
